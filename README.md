# Conjunto de Herramientas Web
Coleccion de herramientas utiles para desarrolladores y usuarios web desarrollada con tecnologías modernas y un diseño responsivo 

# Características
- Interfaz moderna con Bootstrap 5
- Temas claro y oscuro intercambiables
- Sistema de favoritos para herramientas frecuentes
- Diseño responsivo para todos los dispositivos
- Herramientas completamente funcionales sin dependencias de servidor

# Herramientas Incluidas
Seguridad y Encriptacion
-Conversor Base64 codifica y decodifica texto en Base64
-Decodificador JWT decodifica tokens JWT y muestra su contenido
-Generador de Hashes genera MD5, SHA-1, SHA-256 y SHA-512
-Generador de Contraseñas crea contraseñas seguras personalizables

# Manipulacion de Imagenes
-Compresor de Imágenes reduce el tamaño de imágenes manteniendo calidad
-Conversor de Imágenes convierte entre diferentes formatos JPG, PNG y WebP

# Herramientas de Texto
-Contador de Texto cuenta caracteres, palabras, parrafos y más
-Formateador JSON valida, formatea y minimiza JSON

## Generadores
-Generador UUID crea identificadores únicos universales
-Generador RUT genera RUTs chilenos válidos

# Tecnologias Utilizadas
-HTML5
-CSS3
-JavaScript
-Bootstrap 5
-Bootstrap Icons

# Instalacion y Usar
Clona este repositorio
Abre "index.html" en tu navegador web
Y puede usar todas las herramientaas

# Personalizacion
Temas el proyecto incluye soporte para temas claro y oscuro
Favoritos los usuarios pueden marcar herramientas como favoritas para acceso rápido.
